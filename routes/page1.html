This is an vanilla mobile app

Based just on HTML, CSS and JavaScript. No frameworks or build tools are used.

What's the goal?

I am obsessed by the KISS principle and always try to keep it simple.

So I created this app, to show how simple it can be.

If you want to know more about the WHY, read on at here.

Libraries used:

  • Pico CSS (picocss.com)
  • Lucide icons (lucide.dev)
  • reactiveHtml (github)
